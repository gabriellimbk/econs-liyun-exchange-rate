# Exchange Rate Regimes Explorer

This app was moved into the `exchange rate` folder so it can be managed as a standalone project before you publish it to GitHub.

## Run locally

Prerequisites:
- Node.js

Steps:
1. Open a terminal in `exchange rate`
2. Install dependencies with `npm install`
3. Create `.env.local` if needed and set `GEMINI_API_KEY=your_key`
4. Start the dev server with `npm run dev`

The app runs on `http://localhost:3000`.
